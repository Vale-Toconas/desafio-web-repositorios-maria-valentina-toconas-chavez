# Desafío Técnico: Simuladores Interactivos con Lógica Matemática

Plataforma interactiva que incluye un formulario de inscripción inicial, diseñado con una paleta de colores celeste (albiceleste) y tipografías estilizadas.

## Casos Prácticos Incluidos:
1. **Simulador de Transferencia de Calor:** Aplica la Ley de Enfriamiento de Newton utilizando funciones matemáticas nativas como `Math.exp()` y redondeo con `Math.round()`.
2. **Calculador de Combinaciones Complejas:** Evalúa las alternativas lógicas posibles para un sorteo de selección simultánea de dos grupos independientes utilizando una función recursiva propia para el cálculo de factoriales.

## Tecnologías Utilizadas:
- HTML5 Semántico
- CSS3 (Control estricto de Box Model y diseño responsivo)
- JavaScript Limpio (Manipulación del DOM mediante `document.getElementById`)
